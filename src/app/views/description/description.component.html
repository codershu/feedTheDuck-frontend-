Description
This is an interview project for FreshWorks Studio at Victoria, BC, Canada.
Requirement:
A scientist is trying to understand how ducks are being fed in parks around the world. She wants to collect the following information:
  • What time the ducks are fed
  • What food the ducks are fed
  • Where the ducks are fed
  • How many ducks are fed
  • How much food the ducks are fed
Tech Stacks:
  • Frontend: Angular
  • Backend: .Net Core
  • Database: SQL Server
  • Deployed on Microsoft Azure
